DEF CSS

cascading style sheets => feuilles de style en cascade : est un langage permettant de definir la presentation graphique des pages web (html, xml) , son avantage principal est la separation du code source du contenu (HTML) au code source du style (CSS)(ce qui entraine la separation des taches par specialite (redacteur du contenu et integrateur Css))

Feuilles de style en cascade

Les feuilles de style en cascade1, généralement appelées CSS de l'anglais Cascading Style Sheets, forment un langage informatique qui décrit la présentation des documents HTML et XML. Les standards définissant CSS sont publiés par le World Wide Web Consortium (W3C). Introduit au milieu des années 1990,

Histoire du CSS

Le concept de feuille de style est présent dès l'origine du World Wide Web : le premier navigateur web WorldWideWeb permet de mettre en forme les documents à l'aide de ce qui serait aujourd'hui considéré comme une « feuille de style utilisateur »2. De même, les navigateurs Viola en 1992 et Harmony en 1993 recourent à un mécanisme similaire permettant de déterminer le rendu des polices de caractères, des couleurs ou de l'alignement du texte.
